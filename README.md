# Modern Web Calculator

A sleek, responsive, and user-friendly calculator built with HTML, CSS, and JavaScript. This calculator features a modern design with smooth animations and a comprehensive set of mathematical operations.

🔗 **[Live Demo](https://generalcalculator.vercel.app/)**

![Calculator Preview](https://github.com/user-attachments/assets/921b3af9-707e-42e9-8ad9-49f816bf6a6a)

## Features

- 🎨 Modern and responsive design
- 🔢 Basic arithmetic operations (+, -, *, /)
- 🧮 Advanced features:
  - Parentheses support for complex expressions
  - Decimal point input
  - Clear (AC) and Delete (DE) functions
- ⚡ Error handling for invalid operations
- 🎯 User-friendly interface with hover effects
- 📱 Mobile-responsive layout

## Technologies Used

- HTML5
- CSS3 (with custom variables and animations)
- JavaScript (ES6+)
- Google Fonts (Poppins)

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/HumayunK01/calculator.git
   ```

2. Open `index.html` in your web browser.

3. Start calculating!

## Usage

- Click the number buttons to input values
- Use operator buttons (+, -, *, /) for calculations
- Press '=' to see the result
- Use 'AC' to clear the display
- Use 'DE' to delete the last entered character
- Use parentheses for complex expressions

## Features in Detail

### Basic Operations
- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)

### Special Functions
- Clear (AC): Resets the calculator
- Delete (DE): Removes the last entered character
- Decimal Point: Allows decimal number input
- Parentheses: Supports complex mathematical expressions

### Error Handling
- Division by zero protection
- Invalid expression detection
- Clear error messages

## Styling
